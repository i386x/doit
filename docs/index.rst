DoIt!: A Simple Extendable Command Language Documentation
=========================================================

Welcome to DoIt! documentation. DoIt! is a modern command/scripting/programming
language which allows user to redefine itself. This is achieved making DoIt! to
be more language interpreter framework than old-fashioned language interpreter.
In DoIt!, user is capable to

- define new lexical elements, syntax constructs, and semantic rules
- manipulate with source input
- use just-in-time compiler and native code generator services
- use predefined support for object oriented programming
- communicate with host environment in binary way
- and more
